# Identity Governance Templates
This repository contains templates, documentation and examples focused on identity governance within Microsoft Entra ID.

The purpose of this collection is to demonstrate best practice approaches for lifecycle management, access reviews, entitlement management and privileged access governance.

---

## Contents

### Lifecycle Management Templates
Documents that describe the joiner mover leaver process including:

• New starter access flow  
• Role change access update process  
• Offboarding and access removal steps  
• Identity cleanup and review tasks  

---

### Access Review Templates
• Access review planning  
• Reviewer guidance  
• Example access review configuration  
• Reporting and follow up template  

---

### Entitlement Management
• Example access package  
• MyAccess portal walkthrough  
• Approval workflows  
• Access expiry and recertification  

---

### Privileged Access Governance
• PIM onboarding guide  
• Sample privileged role policies  
• Role activation process  
• Monitoring and auditing template  

---

## Skills Demonstrated
• Identity lifecycle planning  
• Role based access principles  
• Access review governance  
• Entitlement management  
• Privileged identity governance  
• Security and compliance alignment
