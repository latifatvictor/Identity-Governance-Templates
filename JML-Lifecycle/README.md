# Joiner Mover Leaver (JML) Lifecycle Templates

This folder includes documentation and templates for designing and automating identity lifecycle processes across the organisation.

### Included:
• Joiner workflow  
• Mover workflow  
• Leaver workflow  
• Identity governance controls  
• Access provisioning guidance  
• Account disablement and revocation template  
• HR to IT integration examples  

### Purpose:
JML ensures employees receive the right access when they join, access changes when their role changes, and full access removal when they leave.
