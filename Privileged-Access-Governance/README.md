# Privileged Access Governance Templates

This folder includes templates and examples related to privileged roles, PIM governance and secure admin access.

### Included:
• PIM eligible assignment template  
• Privileged roles governance policy  
• Admin access justification template  
• Activation approval workflow  
• Re-certification schedule examples  

### Purpose:
Privileged Access Governance ensures admins only access elevated roles when necessary, with auditing, approval and least privilege controls.
